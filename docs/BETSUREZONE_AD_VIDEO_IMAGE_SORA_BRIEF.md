# BETSUREZONE 광고 영상·이미지 제작 브리프 (Sora용)

**서비스**: betsurezone.com — 퀀트급 슈어벳·밸류벳 인텔리전스 플랫폼  
**목적**: 랜딩 페이지·대시보드에 넣을 광고 동영상·광고 이미지 제작  
**도구**: **Sora**(동영상) + 이미지 생성기(DALL·E, Midjourney, Stable Diffusion 등)  
**사용법**: 아래 각 섹션의 **Sora Prompt** / **Image Prompt**를 복사해 Sora(또는 해당 도구)에 붙여 넣어 생성. 필요 시 영어 문장만 살짝 수정해 사용.

## 1. 공통 가이드라인

### 1.1 브랜드 톤
- **키워드**: professional, quant finance, sports analytics, data-driven, real-time, executable opportunities.
- **색감**: 다크 UI, **보라(primary #845adf)**·**초록(성공/기회)** 포인트. 밝은 배경은 피하고, 다크 그라데이션(남색→보라) 권장.
- **분위기**: 신뢰감, 전문성, “실행 가능한 기회만 보여준다”는 메시지.

### 1.2 Sora 사용 시 참고
- **텍스트/로고**: Sora는 글자·로고를 정확히 만들기 어렵습니다. “BETSUREZONE”, “2.04”, “Surebet” 등은 **편집 단계에서 오버레이**하는 것을 권장합니다.
- **해상도·비율**:  
  - 랜딩 히어로/배너: **16:9** (1920×1080 등).  
  - 모바일/스토리: **9:16** (1080×1920).  
- **길이**: 광고용은 **10~20초**가 적당합니다. 프롬프트에 “10 second clip”, “15 second video” 등으로 명시하면 좋습니다.

### 1.3 이미지 사용 시 참고
- **비율**: 랜딩·대시보드 배너는 **16:9** 또는 **3:1** (와이드). 카드/블록용은 **4:3** 또는 **1:1**.
- **해상도**: 1920×1080, 1200×630(OG), 800×800 등 용도에 맞게 리사이즈.

## 2. 광고 영상 시나리오 (Sora)

### 영상 A: 실시간 대시보드 (데이터 흐름)

**용도**: 랜딩 페이지 히어로 배경, 대시보드 소개  
**길이**: 10~15초  
**비율**: 16:9  

**한 줄 요약**: 어두운 공간에서 노트북 화면의 실시간 차트·숫자가 흐르고, 카메라가 화면으로 부드럽게 줌인.

**Sora Prompt:**

```
Professional, cinematic 10-second clip. Dark modern office or desk at night. A laptop screen glows in the center showing a sleek analytics dashboard: real-time line charts, numbers updating, purple and green data points and percentages. Soft slow-motion: digits tick up, a curve draws itself across the chart. Camera slowly pushes in toward the screen. Moody lighting, no people. Quant finance, sports analytics mood. 16:9, 4K style.
```

### 영상 B: 경기장에서 데이터로 (스포츠 × 퀀트)

**용도**: “실시간 기회” 메시지, 스포츠·데이터 결합 강조  
**길이**: 12~18초  
**비율**: 16:9  

**한 줄 요약**: 야경의 스포츠 경기장, 그 위로 홀로그램 같은 숫자·라벨이 떠다니고, 다음 컷에서 실내 대시보드로 전환.

**Sora Prompt:**

```
Cinematic 15-second video. First shot: night stadium, football or basketball arena, crowd lights and field. Transparent holographic-style numbers and labels float above the scene: odds-like digits, subtle “value” and “sure” text. Second shot: cut to indoor, same dark tone. Desk with laptop showing a dashboard with similar numbers and graphs. Smooth transition, professional, data-driven. Purple and green accent lights. 16:9.
```

**비율**: 16:9  

**한 줄 요약**: 다크 UI 위에 여러 북/노드가 원형으로 배치되고, 중앙에서 보라·초록 라인이 퍼져 나가 연결됨.

**Sora Prompt:**

```
Short 10-second clip. Dark UI environment, abstract. Circular arrangement of glowing nodes or icons suggesting different bookmakers. Central hub with purple and green light rays or lines connecting to each node. Clean, tech, fintech style. Slight camera rotation or zoom. No readable text. 16:9, sleek and modern.

**용도**: “일상 속 기회”, 온보딩·감성 톤  
**길이**: 10~15초  
**비율**: 16:9 또는 9:16  

**한 줄 요약**: 새벽/아침 창가, 커피와 노트북. 화면에 대시보드가 켜지고, 알림처럼 새 기회가 떠오르는 느낌.

```

Gentle motion: user scrolls through cards or numbers. Peaceful, productive mood. End with a subtle glow on the screen. No faces in frame. 16:9.

### 영상 E: 숫자·그래프만 강조 (미니멀 데이터)

**용도**: 짧은 루프, 배경 영상, 대시보드 섹션

**Sora Prompt:**

```
Minimal 8-second loopable clip. Dark gradient background (navy to purple). Floating numbers and a single line graph updating in real time. Purple and soft green highlights. No people, no UI text. Abstract data visualization, smooth and calm. 16:9.
```
